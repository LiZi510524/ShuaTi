# ShuaTi 使用说明

ShuaTi 是一个刷题工具集合，主要包含：

- 从学习通/超星提交结果页导出题库 Excel。
- 使用“我爱刷题 Pro”导入题库、刷题、复习错题、收藏题目和同步数据。

## 在线打开

直接访问：

```text
https://lizi510524.github.io/ShuaTi/
```

也可以直接打开 Pro App：

```text
https://lizi510524.github.io/ShuaTi/wo-ai-shuati-pro/
```

## 在 iPhone 上添加到主屏幕

1. 用 iPhone Safari 打开上面的在线地址。
2. 点击 Safari 底部的分享按钮。
3. 选择“添加到主屏幕”。
4. 回到桌面，打开“我爱刷题 Pro”。

如果更新后还是旧界面，先关闭 App，再重新打开一次。

## 从学习通导出题库

### 安装导出脚本

1. 浏览器安装 Tampermonkey / 篡改猴 / 油猴。
2. 新建脚本。
3. 打开 `expert_script/chaoxing-source-export.user.js`。
4. 复制全部内容，粘贴到油猴脚本里并保存。

### 导出 Excel

1. 打开学习通/超星已经提交后的作业详情页。
2. 页面右下角会出现“学习通导题导出”面板。
3. 点击“后台导出 Excel”。
4. 保存导出的 `.xlsx` 文件。

如果页面识别不准，可以点击“导出调试 JSON”，再用这个文件排查。

## Excel 格式

App 支持 `expert_script/导题模板.xlsx` 这种格式：

| 列 | 内容 |
|---|---|
| A | 题干 |
| B | 正确答案 |
| C | 解析 |
| D-J | 选项 A-G |

答案填写示例：

- 单选：`A`
- 多选：`AB`、`ACD`
- 判断：`正确`、`错误`、`对`、`错`、`√`、`×`
- 填空：题干含 `{答案}`，或没有选项

## 导入题库

1. 打开“我爱刷题 Pro”。
2. 进入“题库”页。
3. 在“导入题库”里填写课程、章节和标签。
4. 选择 Excel 文件。
5. 点击“导入为新题库”。

导入后会自动生成一个题库，可以在“我的题库”里管理。

## 刷题

1. 在“我的题库”里点击“开始刷题”。
2. 选择刷题模式：顺序、随机、错题、收藏、未做。
3. 做完一题后点击“提交答案”。
4. 查看正确答案和解析。
5. 点击“下一题”继续。

刷题页点击题号旁边的下拉按钮，可以打开题目目录，直接跳到某一题。

## 错题与收藏

- 答错后可以手动点击“加入错题本”。
- 在题目右上角点击星标可以收藏题目。
- 底部进入“错题”页，可以重练错题。
- 在“刷题”页切换“错题”或“收藏”模式，也可以专项练习。

## 公开题库与云同步

如果只在本机刷题，不需要登录。

如果要发布公开题库或同步数据：

1. 进入“我的”页。
2. 用邮箱登录。
3. 设置用户名。
4. 回到“题库”页。
5. 在题库卡片点击“公开发布”。
6. 其他人可以在“发现”页搜索你的用户名，或输入题库 ID 保存题库。

同步入口在“我的”页，点击“同步题库与记录”即可。

## 备份与恢复

1. 进入“我的”页。
2. 找到“设置与备份”。
3. 点击“导出全部备份”保存 JSON 文件。
4. 需要恢复时，选择备份 JSON 文件导入。

建议定期导出备份，尤其是在清理浏览器数据或更换设备之前。
